Deteksi Ancaman Digital

Menganalisis file, domain, IP, dan URL yang mencurigakan untuk mendeteksi malware dan pelanggaran lainnya, dan secara otomatis membagikannya dengan komunitas keamanan.

Analisis File

Unggah file Anda untuk dipindai. Ukuran file maksimum: 128MB. File yang didukung termasuk PDF, Dokumen Office, Gambar, dll.

Klik untuk memilih file atau seret & jatuhkan file di sini

Analisis URL

Masukkan URL lengkap (misalnya, http://example.com/) untuk memeriksa potensi ancaman.

Pencarian Intelijen Ancaman

Cari berdasarkan hash file (MD5, SHA1, SHA256), domain, alamat IP, atau URL untuk mendapatkan laporan detail.

Dengan mengirimkan data di atas, Anda menyetujui Ketentuan Layanan dan Pemberitahuan Privasi kami, dan untuk berbagi pengiriman Sampel Anda dengan komunitas keamanan. Mohon untuk tidak mengirimkan informasi pribadi apa pun; CyberAware tidak bertanggung jawab atas isi kiriman Anda. Pelajari lebih lanjut.
